# Changelog

## v1.0.0 — Foundation Release

### Added

- First public GitHub release structure.
- Exhaust, intake, turbo and BOV tuning foundation.
- Engine-family balancing notes for key performance platforms.
- Backfire and anti-lag consistency pass.
- Clean documentation for installation and feedback.

### Improved

- Clearer progression between stock, sport and race sound profiles.
- Stronger presence for selected high-performance builds.
- More natural turbo response on supported vehicles.

### Notes

This release is focused on a stable audio base. More vehicle-specific refinements can be added in later builds.
