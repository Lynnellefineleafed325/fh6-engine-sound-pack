# FAQ

## Where do I download the pack?

Use the latest file in GitHub Releases.

## Is this a full vehicle replacement pack?

No. The goal is quality-focused sound refinement and progression tuning, not inflated vehicle counts.

## Can I use it with other audio projects?

Only if the packages do not overwrite the same files. Back up your files first.

## How do I report a sound issue?

Open a sound feedback issue and include:

- Vehicle name
- Upgrade level
- What sounds wrong
- Whether other packages are installed
