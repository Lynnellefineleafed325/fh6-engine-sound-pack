## Overview

**Horizon Engine Audio Suite** is a carefully tuned audio enhancement project for Forza Horizon 6.  
The goal is simple: make vehicle sound progression feel clearer, stronger, and more natural without turning every car into the same aggressive mix.

This repository is designed as a clean download hub. The latest package is always published in **GitHub Releases**, so visitors can open the page and download the newest build quickly.

## What it improves

- Deeper exhaust presence for selected performance builds
- Clearer intake response across upgrade stages
- More noticeable turbo spool and release behavior
- Rebalanced backfire and anti-lag style audio events
- Cleaner engine-family progression for VTEC, rotary, flat-four, V10, V12, 2JZ, RB26, VR38, B58 and more
- More consistent sound behavior between stock, sport and race setups
- Better distance presence for traffic, AI and nearby vehicles

## Installation

1. Download the latest release archive.
2. Extract the archive to a temporary folder.
3. Read `INSTALL.txt` inside the package.
4. Back up any files you plan to replace.
5. Copy the package files into the matching game/mod manager location.
6. Start the game and test with several vehicle classes.

Full guide: [`docs/INSTALL.md`](docs/INSTALL.md)
