# Installation Guide

## Before installing

- Use the latest release package from GitHub Releases.
- Keep a backup of the original files.
- Do not mix several audio packs unless you know which files overlap.

## Steps

1. Download the newest `.zip` from the Releases page.
2. Extract the archive.
3. Open the extracted folder and read the included `INSTALL.txt`.
4. Copy the files to the required location shown in the release package.
5. Launch the game and test several vehicles.

## Clean update

When updating from an older build:

1. Remove files from the previous build if the release notes ask for it.
2. Install the new package.
3. Test stock, sport and race upgrades separately.

## Troubleshooting

If something sounds too quiet, too loud or missing:

- Check that all files were copied correctly.
- Confirm that another audio package is not overriding the same files.
- Reinstall the latest release.
- Open a sound feedback issue with vehicle name, upgrade level and a short description.
